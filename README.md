# SwiftFirstLook

First look Swift as a backend with Vapor.

## Features

|     | Feature                  | Description               |
| --- | ------------------------ | ------------------------- |
| ✅  | Todo (Built in template) | Get, Create, Delete Todos |
| ✅  | Todo Items               | Add checklist todo items  |

## Getting Started

```bash
swift run
```
